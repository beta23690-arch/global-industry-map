# 全球行业关系图 🌐

一个单文件交互式网页：把 12 大行业、细分领域、150+ 全球代表公司用力导向关系图串起来，点击即可下钻查看细分领域、代表公司简介、**实时股价**，并一键跳转财经页面。

## ✨ 功能

- 🕸️ **12 大行业关系网**（ECharts 力导向图）——可拖动 / 滚轮缩放 / 图例筛选，连线标注行业间关系
- 📂 **逐层下钻**：点击行业 → 细分领域 → 代表公司（名称 / 股票代码 / 国家 / 一句话简介）
- 💹 **实时股价 + 涨跌幅**，货币符号自动区分：美股 `$` / A股 `¥` / 港股 `HK$`，每 60 秒刷新
- 🔗 **点击公司卡片** → 跳转对应财经页（股价 / 新闻 / 财报）

## 🚀 使用

直接打开 `industry-map.html` 即可（双击，或部署后访问在线网址）。

**实时股价数据源（混合）：**
- **美股 / 海外 ADR** → [Finnhub](https://finnhub.io)（需免费 API Key，首次打开在顶部填入，仅存于浏览器本地 localStorage）
- **A股 / 港股** → 腾讯财经公开接口（免 Key，开箱即用）

**点击公司跳转：** 美股 → Yahoo Finance；A股 / 港股 → 东方财富个股页。
